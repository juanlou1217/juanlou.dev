# Contracts

这里记录项目的重要契约，用来约束输入、输出、边界和长期稳定的约定。

## 这类文档解决什么问题

- 减少“大家以为是这样”的口头约定
- 让后续改动更容易判断是否破坏已有能力
- 给协作者和 AI agent 一个明确的边界参考

## 典型契约类型

- API 输入输出约束
- MDX frontmatter 约束
- 数据模型字段约束
- 环境变量约束
- 页面路由和 SEO 约束
